# TodoList

Une application web simple pour gérer vos tâches.

## À quoi ça sert ?

Cette application permet de :

- Ajouter des tâches
- Marquer les tâches comme terminées
- Supprimer des tâches
- Filtrer vos tâches (toutes, à faire, terminées)
- Basculer entre mode clair et mode sombre

Vos tâches sont sauvegardées automatiquement dans votre navigateur avec le Local Storage.

## Technologies

- HTML
- CSS
- JavaScript
